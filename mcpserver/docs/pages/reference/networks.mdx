# Networks

The Ammplify MCP server supports both Monad mainnet and testnet.

## Configuration

Set the `AMMPLIFY_NETWORK` environment variable:

```bash
# Mainnet (default)
AMMPLIFY_NETWORK=mainnet

# Testnet
AMMPLIFY_NETWORK=testnet
```

This automatically sets the correct RPC URL, chain ID, and deployed addresses.

## Override Individual Settings

You can override any auto-configured value:

```bash
AMMPLIFY_NETWORK=mainnet

# Override just the RPC (e.g., use a private node)
AMMPLIFY_RPC_URL=https://my-private-monad-rpc.example.com

# Override the addresses file
AMMPLIFY_ADDRESSES_FILE=./addresses/custom.json
```

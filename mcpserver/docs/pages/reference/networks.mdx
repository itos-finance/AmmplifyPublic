# Networks

Ammplify MCP runs on Monad mainnet (chain 143). Everything is pre-configured out of the box.

## Defaults

| Setting | Value |
|---------|-------|
| **Chain ID** | 143 |
| **RPC URL** | `https://rpc.monad.xyz` |
| **Diamond** | `0x5b5e9d616fafccc4865a6c29b6f89ff3aae7c892` |
| **Addresses File** | `addresses/monad-mainnet.json` |

## Overrides

If you want to use your own RPC node:

```bash
AMMPLIFY_RPC_URL=https://my-monad-rpc.example.com
AMMPLIFY_CHAIN_ID=143
```
